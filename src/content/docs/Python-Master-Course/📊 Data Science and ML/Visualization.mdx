### Personalizaci칩n de Gr치ficos
*   Cambiar el estilo de l칤nea: `plt.plot(x, y, linestyle="--")`.
*   Cambiar el color de l칤nea: `plt.plot(x, y, color="red")`.
*   Cambiar el marcador de los puntos: `plt.scatter(x, y, marker="o")`.
* Cambiar el tama침o de los marcadores y el tama침o de la l칤nea: `plt.plot(x,y, linewidth=2)`, `plt.scatter(x, y, s=10)`.
*  Cambiar el rango de los ejes: `plt.xlim(0,10)`, `plt.ylim(0,10)`.
*  A침adir texto en un punto espec칤fico :`plt.text(x,y, "texto")`.
*  A침adir una grilla: `plt.grid(True)`.
*  Cambiar el tama침o del grafico: `plt.figure(figsize=(8,6))`.

```python

plt.show()
```

```python
plt.title("Boxplot de Edades por ciudad")
plt.show()

sns.violinplot(df, x="ciudad", y="edad")
plt.title("Violinplot de Edades por ciudad")
plt.show()

corr = df[["edad", "puntaje"]].corr()
sns.heatmap(corr, annot=True)
plt.title("Mapa de Calor de Correlaci칩n")
```

### Gr치ficos Interactivos

*  `px.line(data, x, y)`: Crea un gr치fico de l칤neas interactivo.
*  `px.scatter(data, x, y)`: Crea un gr치fico de dispersi칩n interactivo.
*  `px.bar(data, x, y)`: Crea un gr치fico de barras interactivo.
* `px.pie(data, values, names)`: Crea un grafico de torta interactivo

```python

data = {
    "nombre":["Juan", "Maria", "Pedro","Ana", "Luis","Sofia"],
    "edad": [30, 25, 40, 18, 22,30],
    "ciudad": ["Madrid", "Barcelona", "Valencia","Madrid", "Valencia", "Barcelona"],
    "estudios": ["bachiller","universitario", "doctorado","bachiller","universitario","doctorado"],
    "puntaje":[8,6,9,5,7,10]
}
df = pd.DataFrame(data)

fig = px.line(df, x="edad", y="puntaje", color="ciudad", title="Lineas de Edades vs Puntaje")
fig.show()

fig = px.scatter(df, x="edad", y="puntaje", color="ciudad", title="Dispersi칩n de Edades vs Puntaje")
fig.show()

fig = px.bar(df, x="ciudad", y="edad", color="estudios", title="Barras de Edades por ciudad y estudios")
fig.show()

fig = px.pie(df, values="edad", names="ciudad", title="Torta de Edades por ciudad")
fig.show()
```
