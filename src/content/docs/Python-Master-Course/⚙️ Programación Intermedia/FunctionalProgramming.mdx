## 1. `map`, `filter`, `reduce` <Badge text="Funciones de Orden Superior" variant="tip" />

Las funciones `map`, `filter` y `reduce` son funciones de orden superior, es decir, reciben otras funciones como argumentos.

### `map(function, iterable)`

Aplica una funci贸n a cada elemento de un iterable y retorna un iterador con los resultados.

```python

cuadrados = map(lambda x: x**2, numeros)

```

### `filter(function, iterable)`
Filtra los elementos de un iterable bas谩ndose en una funci贸n que retorna `True` o `False`, retorna un iterador con los elementos que cumplen la condicion.

```python

print(list(pares)) # [2, 4, 6]
```

### `reduce(function, iterable, initial)`
Aplica una funci贸n de manera acumulativa a los elementos de un iterable, retorna un 煤nico valor. Es necesario importar `reduce` desde el m贸dulo `functools`.


<Aside type="note" title="Nota ">

## 3. Inmutabilidad <Badge text="Datos que No Cambian" variant="success" size="large" />

La inmutabilidad es un concepto central de la programaci贸n funcional que implica que los datos no se pueden modificar despu茅s de su creaci贸n.

### Ventajas de la Inmutabilidad

*   **Simplicidad:**  Reduce la complejidad del c贸digo al evitar efectos secundarios.
*   **Concurrencia:** Facilita el uso de datos en entornos concurrentes sin problemas de sincronizaci贸n.
*   **Predicibilidad:** El comportamiento del c贸digo es m谩s predecible, lo que facilita la depuraci贸n.
### Datos Inmutables en Python
*   Tuplas
*   Strings
*   N煤meros
*   Booleanos
*   frozensets

```python

# tupla[0] = 10 # Esto genera un error, las tuplas son inmutables

# cadena[0] = "M" # Esto genera un error, las cadenas son inmutables
```
Para simular la inmutabilidad en listas y diccionarios, se deben crear nuevas instancias al modificar un valor.
```python

nueva_lista = lista + [4] # genera una lista nueva
print(lista) # [1, 2, 3]
print(nueva_lista) # [1, 2, 3, 4]
diccionario = {"a":1, "b":2}
nuevo_diccionario = {**diccionario, "c":3} # genera un diccionario nuevo
print(diccionario) # {'a': 1, 'b': 2}
print(nuevo_diccionario) # {'a': 1, 'b': 2, 'c': 3}
```
