# Tipos de Datos y Variables en Python: La Base de Todo Programa

En este post, vamos a explorar en detalle los tipos de datos y variables en Python. Estos son los bloques de construcci√≥n fundamentales para cualquier programa. Vamos a cubrir:

## 1. N√∫meros en Python <Badge text="El Mundo Num√©rico" variant="tip" />

Python ofrece varios tipos de n√∫meros, cada uno con sus propias caracter√≠sticas:

*   **`int` (Enteros):** N√∫meros sin parte decimal.

    ```python
    entero = 10
    entero_negativo = -5
    ```

*   **`float` (Decimales):** N√∫meros con parte decimal.

    ```python
    decimal = 3.14
    decimal_negativo = -2.71
    ```

*   **`complex` (N√∫meros Complejos):** N√∫meros con parte real e imaginaria.

    ```python
    complejo = 2 + 3j  # 2 es la parte real, 3 es la parte imaginaria
    ```

*   **`decimal.Decimal` (Decimales de Alta Precisi√≥n):** Para c√°lculos financieros y otros casos que requieren alta precisi√≥n.

    ```python
    from decimal import Decimal

    decimal_alta_precision = Decimal('0.1') + Decimal('0.2') # Evita errores de punto flotante
    ```

### Operaciones con N√∫meros

Python soporta las operaciones aritm√©ticas est√°ndar:

*   Suma (`+`)
*   Resta (`-`)
*   Multiplicaci√≥n (`*`)
*   Divisi√≥n (`/`)
*   Divisi√≥n entera (`//`)
*   M√≥dulo (`%`)
*   Potencia (`**`)

```python

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.333...
print(a // b)  # 3

print(a ** b)  # 1000
```

## 2. Strings y Manipulaci√≥n de Texto <Badge text="Cadenas de Caracteres" variant="note" />

Un string es una secuencia de caracteres. En Python, los strings son inmutables (no se pueden cambiar una vez creados).

```python
cadena_simple = 'Hola mundo'
cadena_doble = "Esto es una cadena"
cadena_multilinea = """
Esto es una
cadena
en varias lineas
"""
```

### M√©todos de Strings

Python ofrece muchos m√©todos para manipular strings:

*   **`.upper()`:** Convierte a may√∫sculas.
*   **`.lower()`:** Convierte a min√∫sculas.
*   **`.strip()`:** Elimina espacios en blanco al inicio y al final.
*   **`.replace()`:** Reemplaza una subcadena por otra.
*   **`.split()`:** Divide la cadena en una lista de subcadenas.
*   **`.join()`:** Une una lista de cadenas en una sola cadena.
*   **`.find()`:** Busca una subcadena y retorna el √≠ndice de inicio.
*   **`.startswith()`:** Verifica si la cadena comienza con una subcadena dada.
*   **`.endswith()`:** Verifica si la cadena termina con una subcadena dada.

```python
cadena = "  Hola, Mundo!   "
print(cadena.upper())     # "  HOLA, MUNDO!   "
print(cadena.lower())     # "  hola, mundo!   "
print(cadena.strip())     # "Hola, Mundo!"
print(cadena.replace("Mundo", "Python")) # "  Hola, Python!   "
print(cadena.split(","))  # ['  Hola', ' Mundo!   ']

lista_palabras = ['Hola', 'mundo']
print(" ".join(lista_palabras)) # "Hola mundo"

print(cadena.find("Mundo")) # 8
print(cadena.startswith(" ")) # True
print(cadena.endswith(" ")) # True
```

### Formateo de Strings

Hay varias maneras de formatear strings en Python:

*   **Concatenaci√≥n:** Usando el operador `+` (menos eficiente).
*   **`%` Formateo (antiguo):** Sintaxis similar a `printf` de C.

    ```python
    nombre = "Ana"
    edad = 25
    print("Hola, %s. Tienes %d a√±os." % (nombre, edad))
    ```

*   **`.format()` M√©todo:** M√°s flexible que la concatenaci√≥n.

    ```python
    print("Hola, {}. Tienes {} a√±os.".format(nombre, edad))
    print("Hola, {nombre_usuario}. Tienes {edad_usuario} a√±os.".format(nombre_usuario=nombre, edad_usuario=edad))
    ```

*   **`f-strings` (Python 3.6+):** La manera m√°s legible y eficiente.

    ```python
    print(f"Hola, {nombre}. Tienes {edad} a√±os.")
    ```

### Expresiones Regulares B√°sicas

Las expresiones regulares son patrones que se usan para buscar y manipular texto. Para usarlas en Python, necesitas el m√≥dulo `re`.

```python
import re
patron = r"Hola"
texto = "Hola, mundo! Hola otra vez!"

resultado = re.findall(patron, texto) # Busca todas las ocurrencias
print(resultado) # ['Hola', 'Hola']

resultado = re.search(patron, texto) # Busca la primera ocurrencia
print(resultado.start()) #0
print(resultado.end())   #4
```

*   **`re.findall()`:** Busca todas las coincidencias del patr√≥n.
*   **`re.search()`:** Busca la primera coincidencia.
*   **`re.sub()`:** Sustituye coincidencias por otra cadena.
*   **`re.split()`:** Divide la cadena en una lista de subcadenas usando un patr√≥n.

<Aside type="caution" title="Precauci√≥n ‚ö†Ô∏è">
    Las expresiones regulares pueden ser complejas al principio, pero son muy poderosas para manipular texto.
</Aside>

## 3. Booleanos y Operadores L√≥gicos <Badge text="Verdadero o Falso" variant="success" size="large" />

El tipo de dato booleano solo tiene dos valores posibles: `True` (verdadero) y `False` (falso).

```python
es_mayor_de_edad = True
tiene_cuenta_bancaria = False
```

### Operadores L√≥gicos
*   `and` (Y l√≥gico): Retorna `True` solo si ambos operandos son `True`.
*   `or` (O l√≥gico): Retorna `True` si al menos uno de los operandos es `True`.
*   `not` (NO l√≥gico): Niega el valor del operando.

```python
edad = 20
tiene_licencia = True

print(edad >= 18 and tiene_licencia)   # True
print(edad < 18 or tiene_licencia)     # True
print(not tiene_licencia)             # False
```

### Operadores de Comparaci√≥n
*   `==` (Igual a)
*   `!=` (Diferente a)
*   `>` (Mayor que)
*   `<` (Menor que)
*   `>=` (Mayor o igual a)
*   `<=` (Menor o igual a)

```python
a = 5
b = 10
print(a == b)  # False
print(a != b)  # True
print(a > b)   # False
print(a < b)   # True
```

## 4. `None` <Badge text="La Ausencia de Valor" variant="note" />

`None` es un tipo de dato especial que representa la ausencia de valor. Es √∫til para indicar que una variable no tiene un valor asignado.

```python

```

<Aside type="tip" title="Cu√°ndo usar None ‚ú®">
    Usa `None` para inicializar variables que a√∫n no tienen un valor espec√≠fico o para indicar que una funci√≥n no retorna nada √∫til.
</Aside>

### Ejemplos de uso
*   Inicializar variables.
*   Indicar que una funci√≥n no retorna un valor expl√≠cito.
*   Verificar si una variable tiene un valor asignado.

```python
def funcion_sin_retorno():
  print("Hola")
  return None;

resultado = funcion_sin_retorno()

if resultado is None:
   print("La funcion no retorno nada")
```
